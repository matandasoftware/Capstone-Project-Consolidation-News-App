# Manual GitHub Upload Process

## Step 1: Prepare Files for Upload
1. Create a new folder called "buroko-kitchen-upload"
2. Copy ALL files from the project EXCEPT:
   - node_modules/ folders
   - package-lock.json files
   - Any .env files

## Step 2: Create GitHub Repository
1. Go to https://github.com
2. Click "New repository"
3. Name it "buroko-kitchen"
4. Add description: "Full-stack food ordering application for Buroko's Kitchen restaurant"
5. Make it Public or Private (your choice)
6. Don't initialize with README (we have one)
7. Click "Create repository"

## Step 3: Upload Files
1. Click "uploading an existing file"
2. Drag and drop all prepared files
3. Write commit message: "Initial commit: Buroko's Kitchen food ordering application"
4. Click "Commit changes"

## Files to Include:
✅ client/ folder (without node_modules)
✅ server/ folder (without node_modules)
✅ .github/ folder
✅ package.json
✅ README.md
✅ .gitignore
✅ setup.bat
✅ QUICK_START.md
✅ STATUS.md
✅ demo.html

## Files to EXCLUDE:
❌ node_modules/ folders
❌ package-lock.json
❌ .env files
❌ Any temporary files
